Política de privacidade

Tratamento de dados do cliente

O Grupo Nexum Altivon utiliza dados pessoais para cadastro, atendimento, processamento de pedidos, emissão de documentos, comunicação autorizada e segurança da operação.

Não vendemos dados pessoais. O compartilhamento ocorre apenas quando necessário para executar compras, pagamentos, entregas, obrigações legais e suporte operacional.

Solicitações sobre dados podem ser feitas pelos canais oficiais de atendimento exibidos no site.
